# VM-COM ‚Äì Modern e-commerce solution

<img src="./assets/screenshot.png" width="500px">

VM-COM is a modern e-commerce application that makes online shopping safe and easy.  
The project utilizes a robust and scalable technology stack, ensuring high performance and security.

On the backend, we use **Spring Boot**, known for its stability and enterprise-level security.  
The frontend is built with **Angular**, **TypeScript**, **Bootstrap**, and **SASS**, delivering a clean and responsive UI.

## üî• Features

‚úÖ **Secure Authentication & Authorization** ‚Äì Users can register, log in, and manage their accounts securely.  
‚úÖ **Product Catalog with Filtering & Searching** ‚Äì Easily browse products using dynamic search and category filters.  
‚úÖ **Shopping Cart & Order Simulation** ‚Äì Users can add products to the cart, but order finalization is disabled as this is a demo.  
‚úÖ **Dockerized Deployment with Nginx** ‚Äì Simplified and scalable deployment using containerized environments.
